Basic plugin to brand the login page and welcome email

== Description ==

* Brand the login page including signup and password reset versions
* Brand and customise the welcome email
* Add a link between login and signup (both ways)
* Hide wp-admin from non-admin users
* Hide the admin top bar from non-admin users
* Change the Signup nav link to the user name and allow them to log out
* Go to the homepage on login not to wp-admin
